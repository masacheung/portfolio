MasaNote

MasaNote is a single-page note-taking app that features a rich-text editor, image uploading, autosaving, organization with notebooks. MasaNote is an Evernote clone, a popular note taking web app.

MasaNote's rich-text editor is built with ReactQuill, a Quill component for React, to create a clean WYSIWYG editing experience. Modifications to the ReactQuill editor CSS styling to the Snow theme and note autosaving.

Technologies: Rails, PostgreSQL, JQuery, React, Redux, SCSS

Triolingo

Triolingo is a single-page flash card app that features a dictionary, definition, example, synonym, audio pronunciation and discussion board.

Users can search vocabulary with up to four different languages, English, Korean, Japanese and Spanish. Dictionary will generated definition, example, synonym and audio pronunciation. Users can also create the flash card to store the information by clicking one button for future use.

Technologies: MongoDB, Express.js, React, Node.js, CSS

Dropping Down

Dropping Down is a one player game and the goal of this game is go down as many stairs as possible. Player will have 10 health point, when player's lose all the health point or drop down to the ground will be "GAME OVER".

The game will randomly generate different "stairs" - "Normal", "Trampoline", "Trap" and "Fake". Different "stairs" will have different effect.

Technologies: JavaScript, NPM, Webpack, CSS
